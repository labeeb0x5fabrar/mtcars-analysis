# mtcars Analysis Project

## Objective
To analyze the mtcars dataset using Python and Pandas.

## Tools Used
- Python
- Pandas
- Jupyter Notebook
- Matplotlib - Seaborn
- GitHub

## Tasks Performed
- Data loading and inspection
- Groupby analysis
- Pivot table creation
- Data mapping
- Visualization
- Insights and conclusion

## Key Findings
- Heavier and powerful cars tend to consume more fuel.
- Manual cars often show better mileage than automatic cars.
- Displacement tend to increase horsepower, which in turn decreases 1/4 mi time.
- Differential ratio doesn't necessarily increase mileage
